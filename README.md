# Cushier

Mesin Kasir di Ponsel Anda!

## Apa itu Cushier?

Cushier adalah aplikasi pengelolaan usaha serta Point of Sale (POS) berbasis cloud yang mudah digunakan tanpa perlu proses yang ribet sehingga cocok untuk bidang usaha apapun.
